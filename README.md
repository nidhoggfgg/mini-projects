## calculator (Rust)

a simple but powerful calculator  
pls see this example

```
>>> 12.34 * 45.67 + 6! / 2^4
608.5678
>>> fun f(a b) = a^x + sin(b) * floor(a*b+5)
>>> x = 4
>>> f(ln(PI) E)
5.003415549553682
>>>
```

**note: it is in dev, some thing will change**
